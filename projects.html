Final Year Project

  • AI-Enabled Smart Agricultural System:
    • It is a cloud hosted SaaS that helps identify and report plant diseases to clients through a React.js based website exposing an API. Client can upload pictures and receive insight from our AI model.
    • Keras & Tensorflow, Pandas libraries used to develop models, algorithms .etc.
    • Matplotlib, Seaborn, Pandas use for data visualization(confusion matrix, accuracy graph .etc.)
    • Back propagation used for iterative best model training through transferred learning with VGG19, AlexNet & GoogleNet
    • PlantVillage dataset from Kaggle with 38 classes (healthy and diseasedplantclassification).
    • We are doing dataset collection for local plants.
    • From 67% to 89% prediction accuracy achieved.
    • K-means algorithm based anomaly detection.
